# 学习笔记

## 语言

- 非形式语言: 语法比较松散
- 形式语言: 语法严格
  - 0型 无限制文法
  - 1型 上下文相关文法
  - 2型 上下文无关文法
  - 3型 正则文法

---

## 产生式(BNF)

- 终结符
- 非终结符


### eg.

``` js
// 四则运算
1 + 2 * 3

// 终结符
// 1. Number
// 2. + - * /

// 非终结符
// 1. MultiExpression
// 2. AddExpression
```

### 结合形式语言

- 0型 无限制文法 
> `? ::= ?`
- 1型 上下文相关文法
> `?<A>? ::= ?<B>?`
- 2型 上下文无关文法
> `<A> ::= ?`
- 3型 正则文法
> `<A> ::= <A>?`

### 其它产生式

`EBNF`、`ABNF`
